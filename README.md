# 🎵 Web Music Player

A simple and modern **web-based music player** built using **HTML, CSS, and JavaScript**.
It allows users to play songs, navigate through a playlist, control volume, and view playback progress.

---

## 🚀 Features

* ▶️ Play / Pause music
* ⏭ Next song
* ⏮ Previous song
* 🎚 Volume control
* 📊 Progress bar
* 🎶 Playlist support
* 💻 Simple and responsive UI

---

## 🛠 Technologies Used

* HTML
* CSS
* JavaScript
